---

## Features

### Visual & Interactive Highlights

| Feature | Description |
|---------|-------------|
| **Custom Cursor** | Animated gold-accent cursor that expands on hoverable elements |
| **Noise Overlay** | Subtle film-grain texture for a premium, tactile feel |
| **Parallax Hero** | Slowly zooming background image with staggered text animations |
| **Scroll Animations** | Fade-up and fade-in reveals as sections enter the viewport |
| **Marquee Strip** | Continuous scrolling brand messaging |
| **Sticky Navigation** | Navbar transforms on scroll with backdrop blur |

### Page Sections

| Section | Content |
|---------|---------|
| **Hero** | Full-screen intro with animated headline, CTA, and stats |
| **Philosophy** | Brand values with visual collage and three core pillars |
| **How It Works** | 4-step process with hover-activated gold accents |
| **Trainers** | 3 trainer cards with image zoom and bio reveal on hover |
| **Pricing** | 3-tier transparent pricing with "Most Popular" featured card |
| **Testimonials** | Client success stories with star ratings and avatars |
| **Waitlist** | Lead capture form with city/goal selection |
| **Footer** | Brand info, social links, and legal navigation |

### Form Fields (Waitlist)

- First Name / Last Name
- Email Address
- Phone Number
- City
- Primary Goal (dropdown)

Form includes basic validation and a success state animation.

## Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, keyframe animations
- **Google Fonts** — Cormorant Garamond (display) + Montserrat (body)
- **Vanilla JavaScript** — Cursor tracking, scroll reveal, form interaction
- **Unsplash** — High-quality placeholder imagery (easily replaceable)
